==========
thunk.py
==========

Introduction
=============
thunk.py is a python library and command line client for
interacting with the thunk.us_ web service.


Usage
======
thunk.py can be imported into any python module::

    from thunk import Thunk

    t = new Thunk()
    t.create()
    t.poke("UID")
    t.destroy()




.. _thunk.us: http://thunk.us
